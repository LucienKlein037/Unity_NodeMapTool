# NodeMapTool

## 介绍

这是Unity中节点式地图生成工具(类似杀戮尖塔地图).你可以使用此工具在Unity中以可视化方式创建和管理节点地图，适用于需要图形化节点结构的游戏项目。

## 安装

- 下载 NodeMapTool.unitypackage 文件。
- 打开你的Unity项目。
- 在Unity中，选择 Assets > Import Package > Custom Package。
- 选择你下载的 NodeMapTool.unitypackage 文件并点击 Open。
- 在弹出的导入窗口中，点击 Import 完成工具的导入。

## 版本

Unity 2022.3.60f1 或更新

## 使用说明

在Example文件夹中选择Example场景,在这里播放并了解它的运行.通常你的逻辑会需要修改Example文件夹中的脚本,比较定制化的内容也可以通过修改Core里的脚本.
